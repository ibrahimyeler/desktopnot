# Asistan App - Flutter

Modern ve kullanıcı dostu bir AI asistan uygulaması.

## Özellikler

- 💬 Modern sohbet arayüzü
- 🎨 Güzel ve kullanıcı dostu tasarım
- 📱 Responsive tasarım
- ⚡ Hızlı ve akıcı kullanıcı deneyimi

## Kurulum

1. Projeyi klonlayın veya indirin
2. Bağımlılıkları yükleyin:
   ```bash
   flutter pub get
   ```
3. Uygulamayı çalıştırın:
   ```bash
   flutter run
   ```

## Kullanım

- Mesajınızı yazın ve gönder butonuna basın
- Asistan otomatik olarak yanıt verecektir
- Şu anda basit bir yanıt sistemi kullanılmaktadır

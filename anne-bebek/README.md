# Anne Bebek Android Uygulaması

Bu basit Android uygulaması ekranda "anne" yazısını gösterir.

## Özellikler

- Ekranda "anne" yazısını gösterir
- Modern Material Design kullanır
- Kotlin ile yazılmıştır

## Gereksinimler

- Android Studio
- Android SDK 24+
- Kotlin

## Kurulum

1. Android Studio'yu açın
2. Bu projeyi import edin
3. Gradle sync işlemini tamamlayın
4. Uygulamayı çalıştırın

## Kullanım

Uygulama açıldığında ekranın ortasında "anne" yazısı görüntülenir.

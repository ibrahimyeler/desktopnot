# HomeSupper Landing Page

Modern, modüler ve SEO uyumlu landing page for HomeSupper - Akşam yemeğini düşünmeyi bırak.

## 🚀 Özellikler

- ✅ Modüler component yapısı
- ✅ SEO uyumlu
- ✅ Responsive tasarım
- ✅ Hızlı yükleme
- ✅ Minimal animasyonlar
- ✅ Modern UI/UX

## 📁 Proje Yapısı

```
homesupper-landing/
├── app/
│   ├── page.tsx          # Ana landing page
│   ├── layout.tsx         # Root layout
│   └── globals.css       # Global stiller
├── components/
│   └── landing/          # Landing page section'ları
│       ├── HeroSection.tsx
│       ├── ProblemSection.tsx
│       ├── SolutionSection.tsx
│       ├── HowItWorksSection.tsx
│       ├── WhySingleMenuSection.tsx
│       ├── WhoIsItForSection.tsx
│       ├── StartingModelSection.tsx
│       ├── BrandPhilosophySection.tsx
│       ├── CTASection.tsx
│       └── Footer.tsx
└── package.json
```

## 🛠️ Teknolojiler

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19, Tailwind CSS 4
- **Icons:** Lucide React
- **Language:** TypeScript

## 📦 Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

3. Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın

## 🏗️ Build

Production için build almak için:

```bash
npm run build
npm start
```
